<h3>Structure</h3>

The popover has three main areas:
<ul>
  <li>Header (optional)</li>
  <li>Content</li>
  <li>Footer (optional)</li>
</ul>

<b>Note:</b> The <code>Popover</code> is closed when the user clicks or taps outside the popover or selects an action
within the popover. You can prevent this with the <code>modal</code> property.
