<h3>Keyboard Handling</h3>
The <code>ui5-select</code> provides advanced keyboard handling. If the <code>ui5-select</code> is focused, you can open
or close the drop-down by pressing <code>F4</code>, <code>ALT+UP</code> or <code>ALT+DOWN</code> keys. Once the
drop-down is opened, you can use the <code>UP</code> and <code>DOWN</code> arrow keys to navigate through the available
options and select one by pressing the <code>Space</code> or <code>Enter</code> keys. <br />
