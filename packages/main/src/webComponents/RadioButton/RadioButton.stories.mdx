<h3>Keyboard Handling</h3>
Once the <code>RadioButton</code> is on focus, it might be selected by pressing the Space and Enter keys. <br />
The Arrow Down/Arrow Up and Arrow Left/Arrow Right keys can be used to change selection between next/previous radio
buttons in one group, while TAB and SHIFT + TAB can be used to enter or leave the radio button group. <br />
<b>Note:</b> On entering radio button group, the focus goes to the currently selected radio button.
