<h3>Usage</h3>

The Duration Picker is used for input of time. Users are able to select hours, minutes and seconds. The user can enter a
time by:
<ul>
  <li>Using the sliders that opens in a popup</li>
  <li>Typing it in directly in the input field</li>
</ul>
<br /><br />
